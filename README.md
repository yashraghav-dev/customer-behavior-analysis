Customer Shopping Behavior Analysis

A Complete End-to-End Data Analytics Project

1. Overview

This project analyzes customer shopping behavior using an end-to-end data analytics workflow.
It demonstrates skills in:

Python (EDA + cleaning)

SQL (PostgreSQL/MySQL/SQL Server)

Power BI dashboarding

Reporting & presentation (Gamma)

2. Dataset

Name: Customer Shopping Behavior Dataset
Format: CSV / Excel

Includes fields such as:

Age, gender

Shopping category

Quantity, price

Shopping mall

Payment method

Date/time

Purchase details

6. How to Run the Project
Prerequisites

Install required Python libraries:

pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2

Steps

Clone the repository

Add dataset to data/ folder

Run the Jupyter Notebook in notebooks/

Load cleaned dataset into SQL database

Execute SQL scripts from sql/ folder

Open the Power BI file (.pbix)

View the PDF report and Gamma PPT
